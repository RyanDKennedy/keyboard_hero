# Syengine

## Description
This is a game engine that I am making to learn xcb, vulkan, ecs pattern, and other technologies.

## Building
    git clone
    cd syengine
    mkdir build
    cd build
    cmake ..
    cmake -DCMAKE_BUILD_TYPE=Release ..
    cmake --build .
    cd ..
The binary should now be inside the bin directory in project root.
